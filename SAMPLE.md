# Markdown Theme Sample

This is a sample document to showcase the features and styling of the **Antigravity Markdown Theme**.

## Text Formatting

You can use **bold text**, *italic text*, or ~~strikethrough text~~.
You can also use `inline code` for highlighting small snippets.

## Alerts

> [!NOTE]
> This is a NOTE alert. Useful for providing additional context or information.

> [!TIP]
> This is a TIP alert. Great for suggesting a better way to do something.

> [!IMPORTANT]
> This is an IMPORTANT alert. Used for crucial information that shouldn't be missed.

> [!WARNING]
> This is a WARNING alert. Warns the user about potential issues.

> [!CAUTION]
> This is a CAUTION alert. Advises against actions that could cause data loss or severe problems.

## Lists

### Unordered List
- Item 1
- Item 2
  - Sub-item A
  - Sub-item B

### Ordered List
1. First item
2. Second item
3. Third item

### Task List
- [x] Completed task
  - [ ] Sub task
    - [x] Sub task
- [ ] Pending task 1
- [ ] Pending task 2
